bdc-in-a-box
=============

.. image:: ./img/tools/bdc-in-a-box.png
    :alt: bdcbox
    :height: 128
    :align: right

This page presents the use of the ``bdc-in-a-box`` tool, a utility that allows rapid configuration and indexing of data, without the need for special access to the BDC project infrastructure. Using ``bdc-in-a-box``, it is possible to automatically create ODC instances with ready-to-use BDC data from any region covered by BDC data products.
